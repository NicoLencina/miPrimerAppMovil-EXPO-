# miPrimerAppMovil-EXPO-

Prácticas del **Taller de Nuevas Tecnologías** — primera aplicación móvil construida con [React Native](https://reactnative.dev/) y [Expo](https://expo.dev/).

## 🚀 Comenzar

### Requisitos previos

- [Node.js](https://nodejs.org/) (versión 18 o superior)
- [Expo Go](https://expo.dev/go) instalado en tu dispositivo móvil (Android o iOS)

### Instalación

```bash
# Instalar dependencias
npm install

# Iniciar el servidor de desarrollo
npm start
```

Luego escaneá el código QR con la app **Expo Go** en tu celular.

### Comandos disponibles

| Comando | Descripción |
|---------|-------------|
| `npm start` | Inicia el servidor de desarrollo (Expo Go) |
| `npm run android` | Abre en emulador Android |
| `npm run ios` | Abre en simulador iOS (requiere macOS) |
| `npm run web` | Abre en el navegador web |

## 📱 Características

- **Contador interactivo**: incrementar, decrementar y resetear un contador con estado React
- **Lista de tecnologías**: muestra las tecnologías utilizadas en el taller
